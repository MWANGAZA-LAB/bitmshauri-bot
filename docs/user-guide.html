Getting Started

First Steps

  1. Find the Bot: Search for @BitMshauriBot on Telegram
  2. Start the Bot: Send /start to begin
  3. Language Detection: The bot will automatically detect your preferred language
  4. Welcome Message: You'll receive a welcome message with available features

Pro Tip

Use /help anytime to see all available commands and features.

Basic Commands

/start Basic

Initialize the bot and begin your Bitcoin learning journey

Example: /start
/help Basic

Display comprehensive help menu with all available commands

Example: /help
/language Settings

Switch between Swahili and English languages

Example: /language
/profile User

View your learning progress and achievements

Example: /profile

Troubleshooting

Common Issues

Bot not responding

Solution: Try sending /start again or restart the conversation

Commands not working

Solution: Make sure you're typing commands exactly as shown (case-sensitive)

Price data not updating

Solution: Wait a few minutes and try /price again

Language not changing

Solution: Use /language and select your preferred language
